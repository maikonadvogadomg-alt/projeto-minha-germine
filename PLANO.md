# PLANO DO PROJETO

## 1. RESUMO EXECUTIVO
Este projeto é uma aplicação web que visa oferecer um assistente jurídico utilizando inteligência artificial. A aplicação permite a interação com o usuário através de um chat e fornece informações e serviços relacionados a questões jurídicas. A stack tecnológica utilizada é composta por Node.js, Express, React para o frontend, e PostgreSQL (Neon DB) como banco de dados.
