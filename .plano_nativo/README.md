# Plano Inicial para Criação do Aplicativo Nativo

## 1. Objetivo do Aplicativo
Criar um aplicativo nativo que funcione de forma autônoma no celular, utilizando um banco de dados local e permitindo interações eficientes.

## 2. Tecnologias Utilizadas
- **React Native**: Para desenvolvimento do aplicativo nativo.
- **SQLite**: Para gerenciamento de dados localmente.
